# AgentBar V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a first native macOS menu bar app that reads real local Codex usage signals and safely reports Claude Code source availability.

**Architecture:** SwiftPM executable app using SwiftUI `MenuBarExtra` for the popover, AppKit `NSPanel` for the floating HUD, and read-only local data readers. Shared `UsageStore` owns refresh, settings, aggregation, and UI synchronization.

**Tech Stack:** Swift 6.3, SwiftUI, AppKit, ServiceManagement, XCTest, SwiftPM.
